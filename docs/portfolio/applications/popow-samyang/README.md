# 삼양식품 DXT팀 AI Agent Builder 지원

## 직무 정보

### 직무명
AI Agent Builder (Popow - 삼양식품 DXT팀)

---

## 내 경험과의 연결점

### 1. AI 에이전트 개발 경험
- **AI 아바타 플랫폼**: AutoGen으로 다중 AI 오케스트레이션 시스템 구축
- **지존소프트 n8n**: LLM 기반 품목 매칭 에이전트 개발
- **목원푸드**: Claude API 활용 대화형 간편주문 시스템

### 2. 워크플로우 자동화 전문성
- n8n 워크플로우 설계 및 개발 (4개 핵심 워크플로우)
- Claude API, OpenAI API, Google Vision API 통합 경험
- 웹훅 기반 자동 트리거 시스템 구축

### 3. Full Stack 개발 역량
- React/TypeScript 프론트엔드
- Node.js/FastAPI 백엔드
- PostgreSQL/MSSQL 멀티 DB 아키텍처

### 4. 정량적 성과
- 주문 처리 시간 98% 단축
- AI 품목 매칭 정확도 95%
- 일일 처리 가능량 10배 증가

## 지원 포인트

1. **AI 오케스트레이션**: 다중 LLM 연동 및 워크플로우 제어 경험
2. **실무 적용**: B2B 식자재 유통에서 실제 운영 중인 AI 시스템 개발
3. **Full Stack**: 기획부터 개발, 배포까지 전 과정 독립 수행 가능
4. **문서화**: 21개 기술 문서 작성으로 지식 전달 체계 구축

# 유한킴벌리 e물류기획 지원

## 직무 정보

### 직무명
e 물류기획담당자 (e-Logistics planner)

### 주요 역할과 책임
이커머스의 주문, 재고관리, 운송, 고객배송까지 물류 전반의 공급망이 원활하게 운영될 수 있도록 프로세스 설정하고 전략을 수립

- 비즈니스 확대, 물류센터 전략 수립, 배송 차별화, ESG 전략수립 등 프로젝트 기획 및 이행
- 주문관리시스템(OMS), 재고운영시스템(WMS), 차량및배차관리시스템(TMS), SAP 시스템 개선 및 마스터관리
- 데이터분석 기반 최적화된 영업활동 의사결정 인사이트 제공

### 필요 역량
- 창의적인 사고를 통한 물류 기획 및 비즈니스 확대
- 공급망관리 또는 이커머스 물류 관련 지식
- 물류 및 시스템 개발 프로젝트 경험
- 데이터 분석 관련 툴과 소프트웨어 활용 경험
- 통계적 지식과 데이터 해석 능력

### 성장 포트폴리오
- 이커머스 비즈니스 전반적인 업무 스펙트럼 경험
- 이커머스 전문가, 전략 기획, 프로세스 매니저
- 시스템적 사고 함양
- 공급망관리(SCM) 전체를 총괄하는 리더로 성장

---

## 내 경험과의 연결점

### 1. 시스템 개발 프로젝트 경험
- **지존소프트 n8n 자동화**: 주문 접수 → 품목 매칭 → 발송 처리 → 고객 알림 전 과정 자동화
- **목원푸드 시스템**: OMS 성격의 주문 관리 시스템 Full Stack 개발
- **Good Hands**: 멀티테넌트 SaaS 아키텍처 설계

### 2. 데이터 분석 역량
- ADSP, SQLD 자격증 보유
- PostgreSQL 318개 함수 개발로 데이터 레벨 비즈니스 로직 관리
- AI 기반 품목 매칭 정확도 95% 달성

### 3. 프로세스 자동화 경험
- n8n 워크플로우로 업무 처리 시간 98% 단축
- LLM 기반 지능형 매칭 시스템 설계
- 수작업 대비 일일 처리량 10배 증가

### 4. 물류 도메인 이해
- B2B 식자재 유통 업무 자동화 경험
- 발주서 자동 생성, 검수서 출력 시스템 개발
- 거래처/품목 마스터 데이터 관리 시스템 구축

## 지원 포인트

1. **시스템적 사고**: 개발자 관점에서 OMS, WMS 시스템 설계 경험
2. **데이터 분석**: ADSP/SQLD 자격 + 실무 데이터 파이프라인 구축 경험
3. **AI 활용**: LLM 기반 업무 자동화로 정량적 성과 창출
4. **기획력**: 서비스 기획부터 개발까지 전 과정 주도 경험

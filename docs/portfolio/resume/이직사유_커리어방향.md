# 이직 사유 및 커리어 방향

## 현재 상황

- **재직 회사**: 지존소프트 (4명 스타트업)
- **재직 기간**: 2025.07 ~ 현재 (인턴 2개월 → 정규직 5개월)
- **역할**: 웹 메인 개발자 / AI 서비스 기획

---

## 지존소프트에서 얻은 것

### 성장한 부분
- AI 자동화 시스템 **기획부터 개발까지 전 과정** 주도 경험
- n8n 워크플로우 설계 및 LLM 연동 실무 역량
- 매칭 정확도 50% → 97% 달성 (문제 해결 능력)
- 프론트엔드 + 백엔드 + 워크플로우 **Full Stack 경험**
- 바이브 코딩 및 Claude Code 활용 개발 생산성 향상
- B2B 식자재 유통 도메인 이해

### 감사한 점
- 대표님의 신뢰 속에서 자율적으로 일할 수 있었음
- 아이디어를 직접 기획하고 구현할 기회 (자연어 사전 시스템 등)
- 스타트업 특유의 빠른 의사결정과 실행력 경험

---

## 이직을 결심한 이유

### 1. 성장 환경의 확장
- 4명 스타트업에서 **실무 역량**은 충분히 쌓음
- 이제는 **대규모 서비스 운영**, **체계적인 개발 프로세스** 경험 필요
- 다양한 개발자/기획자들과 협업하며 **시야를 넓히고 싶음**
- 코드 리뷰, 기술 토론 등 **성장을 자극하는 환경** 희망

### 2. 역할 불균형
- 기획도 하고 싶지만 **개발 비중이 80% 이상**
- 본인 개발 + 팀원 개발 지원까지 담당하는 상황
- 충분한 기획/설계 시간 없이 바로 개발에 투입
- **기획에 더 집중하고, 깊이 있는 고민 후 체계적인 개발**을 하고 싶음

### 3. 협업 경험의 한계
- 팀원들의 작업 방식이 달라 지원 시 어려움
- 혼자 지침서, Claude 스킬 만들어 통일하려 했지만 한계
- **다양한 사람들과 일하며 협업 방식을 배우고 싶음**

### 4. 보상
- 현재 최저시급 수준
- 담당 업무와 기여도 대비 보상 불일치

---

## 원하는 커리어 방향

### 단기 목표 (1-2년)
- **기획 + 개발을 균형 있게** 할 수 있는 포지션
- 더 많은 사용자가 쓰는 서비스 개발 경험
- 체계적인 개발 프로세스 속에서 협업 역량 강화
- AI/자동화 분야 전문성 심화

### 중장기 목표 (3-5년)
- AI 기술과 비즈니스를 연결하는 **서비스 기획자** 또는 **PM**
- 문제 정의 → 기획 → 개발 → 운영까지 리드할 수 있는 역량
- 팀을 이끌며 주니어를 성장시킬 수 있는 시니어

### 관심 있는 환경
- 대기업/중견기업: 체계적인 프로세스, 대규모 서비스 경험
- 성장하는 스타트업: 빠른 실행력 + 체계 구축 과정 참여
- HR/물류/이커머스: 도메인 전문성 확장
- **회사 규모보다 "성장할 수 있는 환경"이 중요**

## 면접 답변 (요약)

> **"왜 이직하려고 하세요?"**

"지존소프트에서 AI 자동화 시스템을 기획부터 개발까지 주도하며 많이 성장했습니다. 특히 n8n 워크플로우로 매칭 정확도를 50%에서 97%까지 끌어올린 경험은 큰 자산이 됐습니다.

다만 4명 규모의 조직이다 보니, 기획에 충분한 시간을 투자하기보다 개발에 바로 투입되는 경우가 많았고, 다양한 개발자들과 협업하며 배울 기회가 제한적이었습니다.

저는 **기획과 개발을 균형 있게 할 수 있는 환경**에서, **더 많은 사용자가 쓰는 서비스**를 만들어보고 싶습니다. 체계적인 개발 프로세스와 다양한 관점을 가진 팀원들과 함께 성장하고 싶어서 이직을 결심하게 됐습니다."

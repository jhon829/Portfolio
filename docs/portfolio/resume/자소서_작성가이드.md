# 자소서 작성 3단계 가이드

---

## Step 1. 자기소개서 항목 분석

### 질문 유형 파악
- **NCS (공기업)**: 직업기초능력 10가지 중 출제
- **사기업**: 회사별 질문 상이

### 질문 확인 방법
- 채용사이트 (사람인, 잡코리아) → 합격자소서 후기글에서 질문 확인
- 회사 홈페이지 (채용포털사이트) 직접 확인

---

## Step 2. 키워드 및 사례 찾기

### 질문 키워드 예시
- 지원동기, 포부, 갈등해결, 팀워크, 소통, 창의성, 도전, 성실, 윤리, 직무역량 ...

### 사례 소스
- 관련 경험/경력: 인턴, 실습, 알바, 공모전, 경진대회, 특허, 프로젝트, 논문, 동아리, 봉사활동 ...
- **직무와 연관된 사례** 중심으로 선별

---

## Step 3. 실전 자소서 작성

### 구조적 작성법 스킬
- **STAR + LA**: Situation → Task → Action → Result + Learned + Applied
- **헤드메세지 두괄식**: 핵심 메시지를 먼저 제시
- **스토리텔링**: 경험을 이야기 흐름으로 전달
- **ABC 구조**: 서론 → 본론 → 결론
- **단락 구분**: 가독성을 위한 명확한 단락 나누기

### 체크리스트
- [ ] 지원기업 및 직무 맞춤형 자소서 (직무호환성)
- [ ] 인사담당자/면접관 입장에서 이해되도록 작성 (가독성)
- [ ] 블라인드 사항 금지 (공기업 해당)
- [ ] 글자수 제한 확인 (최소 80% 이상 작성)
- [ ] 맞춤법 검사 (띄어쓰기, 오탈자, 반복단어 확인)
